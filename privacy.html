Politique de confidentialité

Click'n Toilette — Application Android par Ti-Clic

🔒 Aucune donnée collectée

Résumé

Click'n Toilette ne collecte, ne stocke et ne transmet aucune donnée personnelle. L'application fonctionne entièrement en local sur votre appareil. Aucun compte n'est requis. Aucun serveur externe n'est contacté pour les données de l'application.

Données de localisation

L'application peut demander l'accès à votre position GPS uniquement pour afficher les toilettes les plus proches de vous sur la carte. Cette information :

  • N'est jamais enregistrée sur l'appareil ou ailleurs.
  • N'est jamais transmise à Ti-Clic ni à un tiers.
  • Est utilisée uniquement en temps réel, le temps de la session.
  • L'accès GPS peut être refusé — l'app reste pleinement fonctionnelle.

Données de navigation / carte

Les itinéraires sont calculés via les API cartographiques d'OpenStreetMap (Leaflet), un service public open source. Consultez la politique de confidentialité d'OpenStreetMap pour plus de détails.

Données stockées localement

Aucune donnée utilisateur n'est enregistrée. La base de données des emplacements (toilettes, restaurants, etc.) est intégrée directement dans l'application et ne nécessite aucune connexion Internet.

Publicité et tiers

Click'n Toilette ne contient aucune publicité et n'intègre aucun SDK de traçage ou d'analyse comportementale (Google Analytics, Firebase, Facebook, etc.).

Enfants

L'application est adaptée à tous les âges. Aucune donnée n'est collectée, y compris pour les utilisateurs de moins de 13 ans.
